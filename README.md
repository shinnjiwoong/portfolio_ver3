 ### 👀 Overview / 개요
 #### 1. 달성목표
  *이 프로젝트를 통해서 달성하고 싶었던 목표는 무엇이었는가?*
  1. 외부의 라이브러리 없이 __순수한 자바스크립트 코드__ 로만 제작한다.
  2. 디자인적으로 최대한 덜어내고, __군더더기 없이 꼭 필요한 정보로만으로__ 흥미로운 웹사이트를 구성한다.


 #### 2. HOW / 구현 방식 
 1. 전체적인 디자인 컨셉은 화면을 4분하여 각 화면마다 다른 데이터를 보여주는 방식으로 결정했다. 
 2. 필수적인 데이터 외에 다른 요소들은 최대한 배제하였기 때문에, 시각적인 동적 요소를 부여하기 위해서 클릭할 때마다 화면의 색이 바뀌는 것으로 보완하고자 했다. 
 3. 색이 바뀌는 부분에 더해서 직접 제작한 음악을 더해 전체적으로 지루해질 수 있는 부분을 보완하고자 했다.
